Open Data Integrator is a subproject of Mural that provides ETL capabilities to extract, transform, and load bulk data between files and databases. It is a true and reliable alternative to proprietary, closed-source extract, transform, and load (ETL) solutions.

Data Integrator provides connectivity to a vast range of heterogeneous and diversified data sources including non-relational data sources. It provides an ETL development and runtime environment that is fully integrated into OpenESB and NetBeans and optimized for handling very large record sets.

Data Integrator is designed to manage and orchestrate high-volume, high-performance data transformation from within the SOA tier. Along with the Java CAPS platform, Data Integrator offers a comprehensive enterprise integration infrastructure. Data Integrator is an enterprise module optimized for ETL functions and can be used for Data Migration.

Some of the core features of Data Integrator include:
  • Sub-component of Mural community that provides high-performance ETL capabilities
  • JBI standard based Service Engine
  • JMX controls for dynamic configuration
  • Intuitive design-time NetBeans Plug-in with drag and drop capabilities
  • BPEL 2.0 invocation support
  • Plugs seamlessly into a SOA Environment
  • Rich GUI support for building joins, source extraction, and target conditions
  • Configurable ETL/ELT strategies
  • Rich operator palette that features data cleansing functions such as standardization, normalization, address parsing, matching, and so on
  • Ability to source data from web pages (HTML tables), RSS Feeds, spreadsheets, and more
